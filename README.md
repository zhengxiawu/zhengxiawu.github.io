# Xiawu Zheng Homepage

This is a lightweight GitHub Pages homepage modeled after a classic academic personal-page style.

## Local Preview

Open `index.html` directly in a browser, or run:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.

## Deployment Notes

For the canonical user homepage URL, the repository should be named:

```text
zhengxiawu.github.io
```

If using the existing old repository `zhengxiawu_blog.github.io`, rename it to `zhengxiawu.github.io` or create a new repository with that exact name, then push these files to the default branch and enable GitHub Pages from the branch root.
